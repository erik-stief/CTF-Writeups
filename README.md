# CTF Writeups 🚩

Welcome to my Capture The Flag (CTF) writeups repository! This is where I document my solutions, methodologies, and scripts for various challenges I've tackled in CTF competitions. 

## About Me

I am a Cyber Security Honours student at Sheridan College with a strong interest in web exploitation, reverse engineering, and cryptography. I use these writeups to track my progress, solidify my understanding of complex concepts, and share my learning with my community.

## Repository Structure

Writeups are organized by year, then by event name, and finally by challenge category. 

```
├── 2026/
│   ├── Event_Name_1/
│   │   ├── Rev/
│   │   │   ├── challenge/
│   │   │   │   ├── challenge.md (Writeup)
│   │   │   │   ├── script.py (if exists)
│   │   │   │   └── tiny_elf (challenge files)
│   │   ├── Pwn/
│   │   └── Crypto/
│   └── Event_Name_2/
└── README.md
```
